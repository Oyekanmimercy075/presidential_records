# presidential_records

Presidential Dataset Cleaning Project

Project Overview

This project focuses on collecting, cleaning, and preparing a presidential dataset for analysis using Python and Microsoft Excel.

The dataset contains information on presidents, political parties, vice presidents, salaries, and record creation/update dates.

Data Collection

- Scraped presidential data from a public website using Python.
- Extracted relevant information including President, Prior, Party, Vice President, Salary, Date Created, and Date Updated.
- Structured the collected data into a tabular format for further processing and analysis.

Data Cleaning Tasks Performed

- Inspected the dataset structure and column contents.
- Created a backup copy of the original dataset.
- Checked for duplicate records and found none.
- Reviewed columns for missing or inconsistent values.
- Standardized name formatting using Proper Case.
- Improved data readability through formatting and alignment.
- Converted the dataset into a structured Excel table for analysis.
